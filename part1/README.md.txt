# Solutions of part 1 exercises

In this part, we will familiarize ourselves with the React-library, which we will be using to write the code that runs in the browser. We will also look at some features of Javascript that are important for understanding React.

# Requirements:

* [node](https://nodejs.org/en/download/)

# Starting the Application

There is one application by folder, to start an application :

```bash
#Heading to the application directory
$cd courseinfo
#Running the application using Vite
$npm run dev
#Or running the application using create-react-app tool
$npm start
```
